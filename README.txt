# Beyond the Keys — מערכת ניהול־על + CRM (גרסה מלאה)
**איך מעלים?**
1) פתח את הריפו שלך GitHub → BeyondTheKeys-Admin
2) Add file → Upload files
3) גרור את *תוכן התיקייה* (לא את ה-ZIP) לשורש הריפו → Commit
4) Settings → Pages → Deploy from a branch → Branch: main → Folder: /root → Save
5) פתח את הכתובת: https://<username>.github.io/BeyondTheKeys-Admin/

**קבצים**
- index.html, style.css, app.js
- favicon.svg
- progress.json, tasks.json
- contacts.json, companies.json, deals.json, activities.json

**שימוש מהיר**
- ייבוא CSV (אנשי קשר/עסקאות) בכפתורים בתוך הדף
- גרירה בין שלבי העסקאות ב-Kanban
- ייצוא JSON: כפתור ייעודי למעלה
- עדכון ערכים מהיר ב-Dashboard (שמירה זמנית בדפדפן)
